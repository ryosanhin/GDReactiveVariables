# GDReactiveVariables
Reactive-like variables for GDScript.

This project is heavily inspired by [R3](https://github.com/Cysharp/R3), a excellent Rx library for C#. Therefore, the API and concept intentionally resemble R3.

I realize this is "reinventing the wheel," but I created this repository mainly for self-study.
